## ⚙️ Configuration

All settings can be set as **environment variables** (Heroku Config Vars, `.env` file, etc.)

### 🔑 Core Settings

| Variable | Default | Description |
|----------|---------|-------------|
| `SESSION_ID` | *(required)* | Your WhatsApp base64 session string |
| `OWNER_NUMBER` | `254701082940` | Your WhatsApp number (no + or spaces) |
| `OWNER_NAME` | `ObedTechX` | Your name shown in bot messages |
| `BOT_NAME` | `𝐇𝐔𝐍𝐓𝐄𝐑 𝐗𝐌𝐃 PRO` | Bot display name |
| `PREFIX` | `.` | Command prefix (`.`, `!`, `/`) |
| `MODE` | `public` | `public` = anyone, `private` = owner only |

### 🛡️ Protection Settings

| Variable | Default | Description |
|----------|---------|-------------|
| `ANTI_DEL_PATH` | `same` | `same` = alert in same chat, `log` = send to owner DM |
| `ANTI_LINK` | `true` | Delete WhatsApp group links in groups |
| `ANTI_LINK_KICK` | `false` | Kick member who posts links |
| `ANTI_BAD` | `true` | Delete bad/offensive words |
| `ANTI_VV` | `true` | Save & forward view-once messages |

### 📝 AutoBio Settings

| Variable | Default | Description |
|----------|---------|-------------|
| `AUTO_BIO` | `false` | `true` = start auto-bio changer on boot |
| `AUTO_BIO_INTERVAL` | `30` | How often (minutes) to change bio |

### 👥 Group Settings

| Variable | Default | Description |
|----------|---------|-------------|
| `WELCOME` | `true` | Send welcome & goodbye messages in groups |
| `ADMIN_EVENTS` | `true` | Announce promote/demote events |

### 🤖 Auto Features

| Variable | Default | Description |
|----------|---------|-------------|
| `AUTO_STATUS_SEEN` | `true` | Auto-view all status updates |
| `AUTO_STATUS_REACT` | `true` | Auto-react to status updates |
| `AUTO_STATUS_REPLY` | `false` | Auto-reply to status updates |
| `AUTO_STATUS_MSG` | `SEEN YOUR STATUS...` | Message for status auto-reply |
| `AUTO_REACT` | `true` | React to every incoming message |
| `AUTO_REPLY` | `false` | Enable chatbot auto-reply mode |
| `AUTO_TYPING` | `false` | Show typing indicator |
| `AUTO_RECORDING` | `true` | Show recording status |
| `ALWAYS_ONLINE` | `true` | Always appear online |
| `READ_MESSAGE` | `false` | Auto-read all messages |

### 🎨 Customization

| Variable | Default | Description |
|----------|---------|-------------|
| `MENU_IMAGE_URL` | catbox image | Image shown in menus |
| `DESCRIPTION` | `© ᴘᴏᴡᴇʀᴇᴅ ʙʏ ᴏʙᴇᴅᴛᴇᴄʜ` | Footer text in all messages |

## 🔧 Troubleshooting

<details>
<summary><b>❌ Bot not connecting / SESSION_ID error</b></summary>

- Make sure `SESSION_ID` is set correctly in Config Vars
- The session string should be base64 encoded
- Try regenerating your session by deleting `sessions/creds.json` and restarting
- Check that your WhatsApp account is still linked: WhatsApp → Linked Devices

</details>

<details>
<summary><b>❌ Commands not responding</b></summary>

- Check your `PREFIX` setting (default is `.`)
- Make sure `MODE` is set to `public` if you want anyone to use it
- Check bot logs for errors
- Verify the bot's WhatsApp number is not banned

</details>

<details>
<summary><b>❌ Downloaders failing</b></summary>

- All downloaders have multiple API fallbacks — try again after a few seconds
- Some APIs have rate limits — wait 1-2 minutes before retrying
- For YouTube, make sure the video is not age-restricted or private
- For TikTok, use the full URL not the short `vm.tiktok.com` share link

</details>

<details>
<summary><b>❌ Group events (welcome/goodbye) not working</b></summary>

- Set `WELCOME=true` in your config
- Bot must be an **admin** in the group
- Check that `ADMIN_EVENTS=true` for promote/demote alerts

</details>

<details>
<summary><b>❌ Anti-delete not working</b></summary>

- Run `.antidelete on` in your bot's DM
- Set `ANTI_DEL_PATH=same` to get alerts in the same chat
- Bot needs to be running when the message was originally sent (it stores messages in memory)

</details>

<details>
<summary><b>❌ Bot sleeping on free Heroku/Render</b></summary>

- Use a service like [UptimeRobot](https://uptimerobot.com) to ping your bot's URL every 5 minutes
- Your bot URL is: `https://your-app-name.herokuapp.com`
- Set up a free monitor on UptimeRobot pointing to that URL

</details>
